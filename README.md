### Note:

Snag currently supports only four languages.
- Go
- C 
- C++
- Python

### Building process

1. Clone the repo.
```bash
git clone https://utkarshkrsingh/snag.git $HOME/Downloads/snag
```

2. Build it.
```bash
cd $HOME/Downloads/snag
make build 
```

*Snag is currently only available on Linux.*
